# SWC-PLUGIN-OBSIDIAN

This is Obsidian's [SWC](https://swc.rs/) plugin.

## Local development

### Install dependencies
```bash
yarn && yarn build:local
```

### Run tests
```bash
yarn test:local
```

### Choosing an SWC version
Refer to [plugins.swc.rs](https://plugins.swc.rs/) for swc version compatibility
